# AquaKids - Sistema de Gestión Administrativa

## Descripción Breve
AquaKids es una plataforma web desarrollada para la administración integral de una academia de natación. El sistema permite gestionar nadadores, entrenadores, niveles técnicos, asignación de horarios en grupos, control de asistencias y seguimiento de pagos financieros de forma centralizada y eficiente.

## Listado de Módulos
* Dashboard: Visualización de métricas en tiempo real y alertas médicas.
* Gestión de Nadadores: Registro y administración de alumnos.
* Gestión de Entrenadores: Registro y administración de personal docente.
* Niveles y Grupos: Configuración de niveles técnicos y gestión de jornadas.
* Inscripciones: Vinculación de nadadores a grupos específicos.
* Control de Asistencias: Registro de asistencia diaria.
* Gestión de Pagos: Control financiero y estados de cuenta.

## Herramientas Usadas
* Framework: Laravel (PHP).
* Base de Datos: MySQL.
* Frontend: Blade Templating Engine y Tailwind CSS.
* Control de Versiones: Git y GitHub.
* Servidor Local: XAMPP (Apache/MySQL).
